◈ PATH HUNT — how it works
ButtonAction
NEW GAMEPicks a random start term, BFS-finds a target 3–8 hops away, begins the game
GIVE UPReveals the optimal path (highlighted in gold), shows optimal move count

During a game:

  • The current node pulses neon-green (◉ YOU ARE HERE in tooltip)
  • The target node pulses red (◎ TARGET in tooltip)
  • Reachable neighbors glow amber with "→ REACHABLE — click to move" in tooltip
  • All other nodes are dimmed so you focus on what matters
  • Clicking a non-adjacent node shows "NOT CONNECTED ✗"
  • The HUD shows: START / TARGET / AT (current) / MOVES / BEST

When you reach the target:

  • ★ OPTIMAL! — you matched the shortest path
  • ◆ GOOD — within 2 extra moves
  • ◇ DONE — you made it, but could do better

ESC cancels the game and returns to explore mode.
